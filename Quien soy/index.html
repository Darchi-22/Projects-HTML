¿Quien soy?

Soy una persona apasionada por la tecnología y la innovación, con un profundo interés en las tendencias emergentes como la inteligencia artificial, el análisis de grandes volúmenes de datos (big data), y el aprendizaje automático. Mi habilidad para resolver problemas complejos y mi capacidad para trabajar en equipo me permiten abordar proyectos desafiantes de manera efectiva, asegurando que cada solución sea escalable, segura y alineada con los objetivos del negocio.

Las Tecnologías de la Información y la Comunicación (TIC) han transformado mi vida personal y profesional. Me permiten acceder a información, conectarme con personas y realizar tareas de manera más eficiente. Gracias a ellas, he podido aprender y colaborar de formas que antes no imaginaba. Sin embargo, también soy consciente de los retos, como la sobrecarga de información y los riesgos a la privacidad, por lo que trato de usarlas de manera responsable.

- Daniela Archila

Mis proyectos

Proyecto 1


Desarrollar una aplicación móvil que permita a los usuarios organizar y gestionar sus tareas diarias, con funcionalidades como recordatorios, fechas límite, categorías, y sincronización en la nube. El proyecto puede incluir características como la posibilidad de compartir tareas con otros usuarios y asignar prioridades. Tecnologías: React Native, Firebase, SQLite, Push Notifications.


Proyecto 2


Crear una plataforma web donde los usuarios puedan acceder a cursos en línea, seguir clases, hacer pruebas, interactuar con instructores y obtener certificaciones. Incluir funcionalidades como foros de discusión, video en vivo y un sistema de gamificación para motivar a los usuarios. Tecnologías: Node.js, Express, MongoDB, React, WebRTC, AWS.


Proyecto 3


Desarrollar una aplicación que recomiende películas a los usuarios según sus gustos y preferencias. Utilizar técnicas de Machine Learning para analizar el historial de visualización del usuario y predecir qué películas podrían interesarle. La app podría también incluir una interfaz de usuario para calificar y comentar las películas recomendadas. Tecnologías: Python (Scikit-learn, TensorFlow), Flask/Django (backend), React (frontend), APIs de terceros para obtener información sobre películas (por ejemplo, The Movie Database API).


Proyecto 4


Desarrollar un software que permita a las pequeñas empresas gestionar su inventario de productos, controlar stock, realizar pedidos de reposición y generar informes de ventas. El sistema puede incluir funciones de notificación automática cuando el stock está bajo y la integración con plataformas de comercio electrónico. Tecnologías: Python (Django o Flask), PostgreSQL, Bootstrap, API de integración con tiendas en línea (por ejemplo, Shopify).
